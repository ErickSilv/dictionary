# TAD Dicionário

## Descrição

Implementar o Tipo Abstratos de Dado (TAD) Dicionário com base em lista sequencial (i.e. vetor), ordenada e não ordenada. Para a disciplina de EDB I do curso BTI na UFRN.

## Compilando

>make

## Executando

>build/dict
